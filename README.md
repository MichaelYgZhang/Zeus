# Zeus

![zeus](https://raw.githubusercontent.com/MichaelYgZhang/Zeus/master/images/Zeus.png)

##### 简述
- SpringCloud集成项目, 搭建微服务技术体系, 涉及核心技术点如下
    - Zuul 网关, Ribbon + Hystrix
    - OAuth2
    - Eureka
    - Apollo
    - CAT
    - Prometheus
    - MapReduce, Spark, HDFS, Hive, DataWarehouse
